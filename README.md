# OOP-Car-Rental-System
Car Rental System developed in C++ to digitally manage vehicles, customers, and bookings using core OOP concepts.
This project is a console based project whose topic is Car Rental System devloped in C++ using Object Oriented Programming. The main purpose of my project is to simplify and organize the process of renting cars by managing customer details,vehicles and their booking records in a structured way. Completely changing the tradtional ways in which all this was done manually which would often lead to confusion , double bookings, calculation errors and difficulty in maintaining records. Our projects goal is to solve all these issues digitaly using proper OOP principles.
     The system we made allows the users to add and manage car details,register customers , check vehicle availabilityy, calculate rental charges , genrate bills. while devloping this project we are ensuring that we use all the important OOP concepts in order to make our project smooth and fast. Topics such as multiple inheritance,runtime polymorphism,templates and exception handling are widely used to make our project better and easy to use.
     This project is equally divided among three members to ensure collaborative devlopmentand proper version control ussing GitHub with regular commits. This project not only helped us strengthen our understandingof C++ programming but also gives us a practical experience in building a structured and modular software system.
